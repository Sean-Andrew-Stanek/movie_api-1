Project Documentation

Build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

API Endpoints

  • Return a list of ALL movies

    Endpoint URL: http://localhost:8080/movies

    Method Type: GET

    Query Parameters: None

    Expected Body: None

    Response Data Format: JSON

    High-level Description: Returns a list of top 10 movies.

  • Return data about a single movie by title

    Endpoint URL: http://localhost:8080/movies/:title

    Method Type: GET

    Query Parameters: None

    Expected Body: None

    Response Data Format: JSON

    High-level Description: Returns data about a movie based on its title.

  • Return data about a genre by name

    Endpoint URL: http://localhost:8080/genres/:genre

    Method Type: GET

    Query Parameters: None

    Expected Body: None

    Response Data Format: JSON

    High-level Description: Returns data about a genre based on its name.

  • Return data about a director by name

    Endpoint URL: http://localhost:8080/directors/:director

    Method Type: GET

    Query Parameters: None

    Expected Body: None

    Response Data Format: JSON

    High-level Description: Returns data about a director based on their name.

  • Allow new users to register

    Endpoint URL: http://localhost:8080/users

    Method Type: POST

    Query Parameters: None

    Expected Body: JSON object containing user information (e.g., username, password)

    Response Data Format: Plain text or JSON

    High-level Description: Allows new users to register.

  • Allow users to update their user info (username)

    Endpoint URL: http://localhost:8080/users/:id

    Method Type: PUT

    Query Parameters: None

    Expected Body: JSON object containing updated user information (e.g., new username)

    Response Data Format: Plain text or JSON

    High-level Description: Allows users to update their username.

  • Allow users to add a movie to their list of favorites

    Endpoint URL: http://localhost:8080/users/:userId/favorites

    Method Type: POST

    Query Parameters: None

    Expected Body: JSON object containing movie information (e.g., movie ID)

    Response Data Format: Plain text or JSON

    High-level Description: Allows users to add a movie to their list of favorites.

  • Allow users to remove a movie from their list of favorites

    Endpoint URL: http://localhost:8080/users/:userId/favorites/:movieId

    Method Type: DELETE

    Query Parameters: None

    Expected Body: None

    Response Data Format: Plain text or JSON

    High-level Description: Allows users to remove a movie from their list of favorites.

  • Allow existing users to deregister

    Endpoint URL: http://localhost:8080/users/:userId

    Method Type: DELETE

    Query Parameters: None

    Expected Body: None

    Response Data Format: Plain text or JSON

    High-level Description: Allows existing users to deregister.
